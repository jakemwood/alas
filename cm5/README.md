# Alas IO Board

A custom Raspberry Pi 5 Compute Module board for Alas

Forked from the open source Raspberry Pi 5 Compute Module IO board

Modified by [Jurgen Bayer](https://www.fiverr.com/jurgenexpert/design-pcb-for-raspberry-pi-compute-module-cm4-cm5-carrier-board-cm5-io-board)

# File structure

`AlasIO` - the KiCAD files that are relevant.
`Logo Footprints` - the KiCAD footprints for the Ridgeline Radio and ALAS logos

All others are earlier attempts, revisions, drafts, etc.
